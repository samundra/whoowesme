## Introduction

- Backend for Whoowesme
- URL : `api.whoowesme.local`

## Description

This repo is based on [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Database & ORM
- [Postgres](https://www.postgresql.org/)
- [TypeORM](https://typeorm.io/#/)
